# 유용상 딥러닝 기반 무선 네트워크 최적화 기법 개발

본 프로젝트는 딥러닝을 활용하여 무선 네트워크 환경에서 발생하는 트래픽 패턴을 분석하고,  
이를 기반으로 네트워크 자원을 자동으로 할당하는 최적화 기법을 연구·구현하는 것을 목표로 합니다.

---

## 📌 트래픽 기반 자동 자원 할당 기법

무선 네트워크 환경에서는 시간대, 사용자 수, 서비스 유형에 따라 트래픽 변동이 빈번하게 발생합니다.  
본 프로젝트에서는 이러한 트래픽 변화를 **딥러닝 모델을 통해 예측**하고,  
예측 결과를 바탕으로 네트워크 자원을 자동으로 분배하는 기법을 제안합니다.

주요 특징은 다음과 같습니다.

- 과거 트래픽 데이터를 기반으로 한 **트래픽 예측**
- 예측 결과에 따른 **동적 자원 할당**
- 소규모 무선 환경에서도 적용 가능한 **경량화 모델 구조 고려**
