# 유용상 딥러닝 기반 무선 네트워크 최적화 기법 개발

본 프로젝트는 딥러닝을 활용하여 무선 네트워크 환경에서 발생하는 트래픽 패턴을 분석하고,  
이를 기반으로 네트워크 자원을 자동으로 할당하는 최적화 기법을 연구·구현하는 것을 목표로 합니다.

---

## 📌 트래픽 기반 자동 자원 할당 기법

무선 네트워크 환경에서는 시간대, 사용자 수, 서비스 유형에 따라 트래픽 변동이 빈번하게 발생합니다.  
본 프로젝트에서는 이러한 트래픽 변화를 **딥러닝 모델을 통해 예측**하고,  
예측 결과를 바탕으로 네트워크 자원을 자동으로 분배하는 기법을 제안합니다.

주요 특징은 다음과 같습니다.

- 과거 트래픽 데이터를 기반으로 한 **트래픽 예측**
- 예측 결과에 따른 **동적 자원 할당**
- 소규모 무선 환경에서도 적용 가능한 **경량화 모델 구조 고려**

## 🧠 시스템 구성 개요

본 프로젝트의 시스템은 무선 네트워크 트래픽 패턴을 입력으로 받아,
딥러닝 기반으로 네트워크 자원을 자동 최적화하는 구조로 설계된다.
전체 시스템은 크게 데이터 수집 단계, 딥러닝 기반 분석 단계, 자원 할당 단계로 구성된다.

---

1️⃣ 트래픽 데이터 수집 및 전처리 모듈

무선 네트워크 환경에서 발생하는 트래픽 정보를 주기적으로 수집하여
딥러닝 모델 학습 및 추론에 적합한 형태로 전처리한다.

- 시간 단위별 트래픽 소비량(패킷 수, 전송량 등) 수집

- 노드(단말) 또는 서비스 단위 트래픽 집계

- 이상치 제거 및 정규화 처리

- 최근 트래픽 흐름을 반영하기 위한 시계열 데이터 구성

이를 통해 네트워크의 현재 상태를 정량적으로 표현한 입력 데이터를 생성한다.

---

2️⃣ 딥러닝 기반 트래픽 패턴 분석 모듈

전처리된 트래픽 데이터는 딥러닝 모델에 입력되어
네트워크 트래픽의 시간적 변화 및 사용 패턴을 학습한다.

- 과거 트래픽 소비 패턴을 기반으로 한 네트워크 상태 분석

- 트래픽 집중, 분산, 급증과 같은 패턴 인식

- 소규모 무선 환경에서도 동작 가능하도록 경량 딥러닝 모델 구조 적용

본 단계에서 딥러닝 모델은 단순한 수치 계산이 아닌,
네트워크 상태 전반을 반영한 특징(feature)를 추출하는 역할을 수행한다.

---

3️⃣ 자동 자원 할당 및 최적화 모듈

딥러닝 모델의 출력 결과를 기반으로
네트워크 자원을 동적으로 분배하는 자원 할당 로직을 수행한다.

- 트래픽 소비량 및 패턴을 고려한 자원 분배 비율 결정

- 특정 노드 또는 구간에 대한 자원 과다 집중 방지

- 실시간 네트워크 상태 변화에 따른 자원 재조정

이를 통해 현재 네트워크 트래픽 상황에 적합한 자원 배분이 가능하도록 한다.

---

4️⃣ 시스템 통합 및 시각화 모듈

전체 시스템은 하나의 파이프라인으로 통합되어 동작하며,
자원 할당 결과와 네트워크 상태를 사용자에게 직관적으로 제공한다.

- 트래픽 변화에 따른 자원 할당 결과 시각화

- 시간대별 네트워크 부하 변화 확인

- 실험 및 분석을 위한 로그 기록 기능

해당 모듈을 통해 시스템의 동작 결과를 검증하고 분석할 수 있다.
